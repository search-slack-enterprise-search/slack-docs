# setup-enterprise-search
## 知りたいこと
[Slack エンタープライズ検索を設定および管理する | Slack](https://slack.com/intl/ja-jp/help/articles/39044407124755-Slack-%E3%82%A8%E3%83%B3%E3%82%BF%E3%83%BC%E3%83%97%E3%83%A9%E3%82%A4%E3%82%BA%E6%A4%9C%E7%B4%A2%E3%82%92%E8%A8%AD%E5%AE%9A%E3%81%8A%E3%82%88%E3%81%B3%E7%AE%A1%E7%90%86%E3%81%99%E3%82%8B)

上記ページから、Enterprise Searchの有効化の方法をまとめて欲しい。
特にOrgやワークスペースで有効化するだけではエンドユーザーは使えないようなので、有効化の手段をしりたい。

## 目的
Enterprise Searchの有効化では単にワークスペースで有効化するだけではダメみたいなので、詳しい手順を知りたい。

## 調査サマリー

Slack Help Center の日本語版ページ（WebFetch で取得成功）から、Enterprise Search の有効化手順を確認した。有効化は以下の6ステップからなり、エンドユーザーが使えるようになるまでに Org 管理者側の複数の設定が必要。

### 有効化の全体フロー

1. **開発者による App の設定**（開発者の作業）
   - App マニフェストに `features.search` オブジェクトを追加
   - `org_deploy_enabled: true` を設定
   - `function_executed` イベントを購読
   - 検索結果を返すカスタムステップ関数を実装

2. **Org 全体でエンタープライズ検索を有効化**（Org 管理者の作業）
   - 「ツールと設定」→「オーガナイゼーションの設定」→「設定」→「エンタープライズ検索」
   - 「編集」→「オーガナイゼーション全体でエンタープライズ検索を有効にする」にチェック→保存
   - 前提: Enterprise+ または Enterprise Grid プランが必要

3. **App のオーグレベルインストール**（Org 管理者の作業）
   - api.slack.com/apps の UI、OAuth フロー、または Slack CLI でインストール
   - この時点ではワークスペースではまだ使えない

4. **ワークスペースへの App 追加**（Org 管理者の作業）
   - 管理ダッシュボード → Integrations → Installed apps から対象ワークスペースへ追加

5. **データソースの設定とエンドユーザーへの有効化**（Org 管理者の作業）
   - 「エンタープライズ検索」ページからデータソースの「設定する」をクリック
   - 「使用できる人」の「編集」で IDP グループまたは特定メンバーを指定
   - AI 検索時のアクセス許可を設定して保存
   - これが「Org Admin によるエンドユーザーへの有効化」に相当すると考えられる

6. **エンドユーザーによる個人アカウントの連携**（エンドユーザーの作業）
   - Org 管理者の設定後、エンドユーザーが個人アカウントを連携
   - 連携後は検索結果に外部ソース（GitHub, Google Drive など）のコンテンツが表示される
   - エンドユーザーは個別に無効化も可能（サイドバーで右クリック → Disable）

### 補足

- ステップ 5 の「使用できる人」設定が enterprise-search-access-control.md に記載の「Org Admin によるエンドユーザーへの有効化」に該当すると推定されるが、完全な確証は取れていない
- エンドユーザーが個人アカウントを連携する具体的な UI 手順は今回のページには記載なし

## 完了サマリー

Slack Help Center 日本語版ページから Enterprise Search の有効化手順を取得した。
Org 全体の有効化（管理画面でのチェック）→ App インストール → ワークスペース追加 → データソース設定＆使用できる人の指定 → エンドユーザーによる個人アカウント連携、という 6 段階のフローを確認。
「ワークスペースで有効化するだけではエンドユーザーは使えない」問題は、データソースの「使用できる人」設定（ステップ 5）と、エンドユーザー自身による個人アカウント連携（ステップ 6）が必要なことで説明される。
